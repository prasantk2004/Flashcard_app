# 📇 Django Flashcard App
A web-based flashcard application designed to help students create, manage, and review study materials efficiently.

## 📌 Project Overview
This project was built using the **Django Web Framework**. It allows users to create digital flashcards for different subjects, helping with active recall and memory retention.

## ✨ Features
* **CRUD Operations:** Create, Read, Update, and Delete flashcards.
* **Categorization:** Organize cards by topic or subject.
* **Responsive UI:** Clean interface built with Django Templates and CSS.
* **Search Functionality:** Quickly find specific cards within your collection.

## 🛠️ Tech Stack
* **Backend:** Python 3.x, Django 5.x
* **Database:** SQLite (Development)
* **Frontend:** HTML5, CSS3
* **Version Control:** Git & GitHub
